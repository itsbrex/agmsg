# agmsg Logo Assets

Generated from `source-original.jpg`.

## Recommended Files

- `logo-full-transparent.png` - main transparent logo for web and documents.
- `logo-full-on-white.png` - main logo flattened on white.
- `logo-mark-transparent.png` - speech-bubble mark only.
- `logo-wordmark-transparent.png` - wordmark only.
- `icons/apple-touch-icon-180.png` - iOS home screen icon.
- `icons/android-chrome-512.png` - Android/PWA icon.
- `icons/favicon-32.png` - common favicon size.
- `social/og-image-1200x630.jpg` - social sharing preview.

## Notes

- Transparent PNGs preserve the original raster artwork.
- Cast shadows outside the logo have been removed.
- The wordmark counters are transparent cutouts.
- The white `ag` letters inside the mark remain visible.
- Use the navy logo on white or light backgrounds for best contrast.
- The SVG files are convenience wrappers with embedded PNG artwork, not redrawn vector paths.
